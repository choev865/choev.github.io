Title: Stories as Networking
Class: Core 2 Studio
Link: Stories as Networking
Brief Description: For this project, I had to create a short story through HTML, CSS, and multiple screen views. I wanted to create a scavenger love hunt for a possible "loved one or partner" through Notion, Google Forms, Spotify, Instagram, Calendly, and Google Maps to convey my story. I wanted to keep the content very general and non-specific to a certain person so anyone who opens it can feel like it was made for them.

Title: Elastic Collection
Class: Core 2 Studio
Link: Elastic Collection
Brief Description: For this project, I was given a set of data from a classmate and I had to create a collection that displays the 50 items in the dataset. The requirements for this project were to filter and display the 50 items in an Airtable collection and make it responsive as well. My collection was vinyls from the years 1960-2000s. I wanted to make it look like folders where you can sort through vinyls and have a retro pop aesthetic.
×
Title: New York City in Data
Class: Core 2 Studio
Link: New York City in Data
Brief Description: For this project, I had to select a set of data from opendata.cityofnewyork.us to create a website that displays the data. My dataset was a collection of New York City Restaurant Inspection Results with the restaurant name, address, score, grade, violation description, etc. I wanted to replicate the restaurant grading signs that are outside and I sorted it by grade A, B, and C.

Title: Media Tunnel
Class: Core 2 Lab
Link: Media Tunnel
Brief Description: For this project, I had to create a story that moves between a desktop and mobile-size browswer. I created a website that made the coffee pot pour into the coffee cup. Once the media queries changed, it creates a full cup of coffee and a list of coffee shops.

Title: Film Strip
Class: Core 2 Lab
Link: Film Strip
Brief Description: For this project, I had to find a scene from a TV show, movie, or video and gather at least 20 screenshots of frames. I had to create a website that contains the screenshots of frames from that scene into a film strip using css grids. I chose the movie Cruella and each frame needed to include a screenshot and dialogue subtitle.
